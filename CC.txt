###################FOSS CLOUD
Open FOSS Cloud in Mozilla by using the ip address 192.168.10.3
Then input username and password in admin and login as admin
Then open WinSCP and put credential host name: 192.168.10.3, user name:
root and password: admin
After in left hand side, try to navigate to the folder path where you kept your
Ubuntu iso file.
Then in the right hand side, navigate to var>virtualization>iso-choosable. After
that drag and drop the iso file from left to right.
After copying the file, go to the foss cloud and navigate to virtual
machine>profile>create. Then create a Linux profile.
After creating that, go to the vm templates>create and you will see your linux
file select that. Then set no. of display as 1 and create vm template.
Then start the Linux vm by clicking on -> button. After that click vm box button.
The Ubuntu vm will be opened
Then set the iso file to hardware by clicking on cd button.

and domain settings in admin section.

admin@123

Click on add option in table.
Then add a user, by inputting your personalized name, username and
password.
After that logout as admin, and click on owncloud in applications section
Then input username and password that you have created and login
